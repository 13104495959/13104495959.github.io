本地环境检测

⛓检测Java环境是否正常

目的: 检测本地Java环境是否正常.
流程: 打开terminal, 执行 java -version命令.

  1. 输入 java -version命令.
  2. 如下图中2所示,环境正常(最好保持本地的版本是最新的).

⛓检测Android Studio是否正常

需要下载Android Studio,并且按照提示全部进行更新.

Android项目创建和配置

⛓Android项目创建和配置配置

目的: 创建Android项目.
流程: 打开Android studio,选择新建项目,进行配置.

  1. 如下图中1所示.
  1. Application name: 工程名字, 安装App到手机上显示的名称.
  2. Company domain: 公司域名标识.
  3. Project location: 项目路径.
  4. Package name: 包名(可以使用默认的,也可以更改).
  5. 是否支持 C++ 或者 Kotlin(根据实际情况选择).
  6. 下一步.
  1. Phone and Tablet: 选择最低支持的版本,目前是4.0.3可以支持100%的设备.
  2. Instant App: 是否使用即时App(本示例使用,也可以不使用).
  3. 点击下一步.
  1. 如下图中1所示,点击Yes进行Instant App SDK安装.
  1. 如下图中1所示, 等待Instant App SDK下载安装完成, 点击下一步.
  1. 配置即时App的公共包名: 使用功能名 + feature, 必须全部小写.
  2. 点击下一步.
  1. Instant App URL Host: 必须和之前填的公司域名一样.
  2. Instant App URL Route Type: 选择Path Pattern.
  3. Instant App URL Route: / + Module name(上面输入的公共包名)(示例:/mainfeature)
  4. Activity Name: 活动名称.(示例:MainActicity).
    Generate Layout File: 是否自动创建Layou文件.
  5. Layout Name: 布局文件名称.(示例:activity_main).
    Backwards Compatibility (AppCompat): 向后兼容模式(建议勾选).
  6. 点击下一步.
  1. 选择需要的Activity模板, 一般使用Empty Activity.
  2. 点击下一步.
