Sourcery安装和配置

⛓使用Cocoapods安装Sourcery.

目的: 使用Cocoapods安装Sourcery.
流程: 检测CocoaPods环境 –> 编辑Podfile文件 –> 下载Sourcery –> 设置Xcode中的Run Script.

  1. 检测CocoaPods环境, 执行 pod --version命令.如下图显示环境正常,否则需要先安装CocoaPods环境.

创建Podfile文件(如果之前有Podfile文件, 不需要执行此步骤).

  1. 进入到项目文件夹(和.xcodeproj同层级).
  2. 执行 pod init命令.
  1. 找到Podfile文件并打开文件.
  1. 指定平台和版本.
  2. Podfile文件中添加pod 'Sourcery'并保存.
  1. 执行pod update命令, 如图显示下载完成.

⛓使用Xcode配置Sourcery Run Script.

目的: 使用Xcode配置Sourcery Run Script.
流程: 选定target–>选择build Phases–>选择Run Script(没有的话请添加),在Run Scriot中添加脚本$PODS_ROOT/Sourcery/bin/sourcery.

  1. 选择项目文件.
  2. 选择指定的Target并选择build Phases.
  3. 添加Run Script(如果添加过,可以直接使用), 并且编辑脚本$PODS_ROOT/Sourcery/bin/sourcery.

Visual Studio Code安装和配置

⛓Visual Studio Code安装.

  1. 官网: https://code.visualstudio.com/.
  2. 点击下载.
  3. 双击下载好的压缩包.
  1. 解压安装成功.
  2. 点击打开,启动app.

⛓Visual Studio Code配置.

  1. 点击插件.
  2. 输入插件名(stencil 和 stencil tools).
  3. 选择对应的插件.
  4. 点击Install下载.
  5. 下载安装完成.

.sourcery.yml配置文件和.stencil模板文件创建和配置

⛓.sourcery.yml配置文件设置.

  1. 点击插件.
  2. 输入插件名(stencil 和 stencil tools).
  3. 选择对应的插件.
  4. 点击Install下载.
  5. 下载安装完成.

⛓.stencil后缀的模板文件设置.

  1. 点击插件.
  2. 输入插件名(stencil 和 stencil tools).
  3. 选择对应的插件.
  4. 点击Install下载.
  5. 下载安装完成.

快捷键隐藏,使用Visual Studio Code创建配置文件.sourcery.yml(此文件是隐藏的) 和 .stencil后缀的模板文件,并进行编码.

根据模板文件生成.swift文件

使用Xcode进行swift文件编码,command + B, 生成模板对应的swift文件(此文件是隐藏的).
